# Pharmacy backend

Run order:
1. Import `pharmacy_db_full_data.sql` into MySQL.
2. Update `src/main/resources/application.properties` if your MySQL password is not `kien124`.
3. In IntelliJ, open this project from `pom.xml`.
4. Use JDK 17.
5. Maven Reload.
6. Run `PharmacyApplication`.
